Time Lord's Memory
Game Info

Press any Key on your keyboard to Start

Game Instructions
Time Lord's Memory is a fun and challenging game that will put your memory skills to the test! Inspired by the iconic British TV series Doctor Who, this game requires you to remember and reproduce a sequence of flashing coloured buttons. With each level, an additional button is added to the sequence, making it increasingly difficult to keep track of the pattern. You must recall and repeat the previous sequence before adding the newly flashed button. But be careful not to press a button too quickly or repeatedly, as the TARDIS flashing inside the button indicates that it has not yet been registered. As you progress through the levels, the background changes to Doctor Who visuals, adding to the excitement and immersive experience. So keep your memory sharp and your fingers quick to achieve the highest score possible! This game is a must-try for all Doctor Who fans and anyone who loves a fun and challenging memory game. Get ready to embark on a journey through time and space with Time Lord's Memory!
